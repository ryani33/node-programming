What is FC API

I provide 3 calculate funcs in this website, which are all related to our financial life. The first one is to calculate the amount of money you will have in your retirement fund. The second one is to calculate the end result of a sum of money that you invested. The last one is to calculate how many months it will take to pay off a loan.

API Route Instructions

1. perMonthRetirementSavings API Route

Variables years perMonth interestRate
Description How many years until retirement The amount saving per month The yearly interest rate of investment
Request I 3 5 0.1
http://localhost:3000/api/perMonthRetirementSavings?years=3&perMonth=5&interestRate=0.1
Response I {"status":"success","result":210.65001466339126}
Request II 3 5 a
http://localhost:3000/api/perMonthRetirementSavings?years=3&perMonth=5&interestRate=a
Response II {"status":"error","message":"Invalid parameter (yearlyInterestRateOfInvestment)!"}

2. investedAmount API Route

Variables years initial interestRate
Description How many years you are investing The initial amount The yearly interest rate of investment
Request I 3 100 0.1
http://localhost:3000/api/investedAmount?years=3&initial=100&interestRate=0.1
Response I {"status":"success","result":121.00000000000003}
Request II 3 100 N/A
http://localhost:3000/api/investedAmount?years=3&initial=100
Response II {"status":"error","message":"You need provide all three parameters!"}

3. loanPayoff API Route

Variables monthlyAmount loanAmount interestRate
Description Monthly payment amount The initial loan amount The yearly interest rate of load
Request I 10 250 0.1
http://localhost:3000/api/loanPayoff?monthlyAmount=10&loanAmount=250&interestRate=0.1
Response I {"status":"success","result":29}
Request II 3 5 -0.1
http://localhost:3000/api/loanPayoff?monthlyAmount=10&loanAmount=250&interestRate=-0.1
Response II {"status":"error","message":"Invalid parameter (yearlyInterestRateOfLoan)!"}
